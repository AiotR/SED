# Ariketa Berria Sortzeko Skill-a

Gaitasun (Skill) hau erabiltzaileari ariketa berri bat prestatzeko erabiltzen da.

## Pausoak:
1. Galdetu erabiltzaileari zein izan behar den ariketaren izena (adib: "5_Praktika" edo "Hasierako_Proba").
2. Exekutatu sisteman: `cp -r Txantiloia "Nire ebazpenak/{IZENA}"`
3. Ireki `Nire ebazpenak/{IZENA}/src/main.c` fitxategia.
4. Fitxategiaren goiburuan agertzen den izenburua edo deskribapena eguneratu ariketaren izen berriarekin.
5. Esan erabiltzaileari dena prest dagoela Keil programan proiektua (uvproj) irekitzeko eta bertan C kodea garatzen hasteko.
